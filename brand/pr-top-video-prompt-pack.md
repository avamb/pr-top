# PR-TOP — Video Prompt Pack

## Purpose

This file contains ready-to-use prompts for AI-assisted video generation, storyboard expansion, visual exploration, and motion direction.

Use `PR-TOP` as the only public-facing brand name.

If older materials mention `PsyLink`, treat it as a legacy/internal name and do not use it in generated public-facing outputs.

## Global Brand Lock

Apply these rules to every prompt:

- brand: `PR-TOP`
- meaning: `Practice on Top`
- audience: psychologists, therapists, counselors, small therapy centers
- tone: calm, professional, respectful, human, structured
- category: AI-powered practice assistant
- framing: support for therapists, not replacement
- design language: soft-tech, modern health-tech, premium minimal

## Visual Lock

Use:

- deep ink `#163A43`
- therapy teal `#1F8A83`
- soft sage `#A8C9BE`
- mist `#F4F7F6`
- warm sand `#ECE6DD`
- rounded UI
- clean typography
- high whitespace
- subtle motion

Avoid:

- neon
- cyberpunk
- generic startup blobs
- brain icons
- medical cross
- chatbot caricatures
- hype SaaS energy
- robotic-human replacement imagery

## Master Creative Prompt

```text
Create a calm, premium presentation video concept for PR-TOP, an AI-powered practice assistant for psychologists and therapists.

The video must feel therapist-first, not AI-first.
It should communicate reduced routine, continuity between sessions, structured support, and therapist control.

The visual language should be modern health-tech / therapist-tech:
clean, minimal, soft-tech, structured, calm, credible, with rounded UI surfaces and generous whitespace.

Use the following palette:
- Deep Ink #163A43
- Therapy Teal #1F8A83
- Soft Sage #A8C9BE
- Mist #F4F7F6
- Warm Sand #ECE6DD

Do not use PsyLink as a public brand name.
Use PR-TOP only.

Avoid hype, cyberpunk AI aesthetics, fake testimonials, aggressive startup energy, and any framing that suggests replacing therapists.
```

## Prompt: Full 75s Video Concept

```text
Generate a 75-second presentation video concept for PR-TOP.

Goal:
Introduce PR-TOP as a calm AI-powered practice assistant for psychologists and therapists.

The video should communicate:
- this is built for therapists
- it reduces routine
- it supports continuity between sessions
- it keeps care human
- the therapist stays in control

Structure:
1. emotional opening about the hidden weight of private practice
2. visible workflow burden: notes, follow-ups, reminders, client context
3. introduction of PR-TOP as a calm support system
4. value pillars: continuity, communication, less admin, control
5. trust and therapist control
6. final CTA

Visual style:
calm, premium, minimal, modern health-tech, structured, soft-tech

Output:
- scene-by-scene concept
- key visuals
- recommended transitions
- title card ideas
- end card options
```
